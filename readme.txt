1.文字の打ち方
	十字キーの操作と、ABを組み合わせて文字を打ちます。
	
	(例)
	1.上を押したまま、Aを押して離す
		-> b
		.. これが基本
		
	2.上を押してから、上右にする、次にAを押して離す
		-> c
		.. 斜め右にすると、次の文字
		
	3.上を押してから、上左にする、次にAを押して離す
		-> a
		.. 斜め左にすると、前の文字
		
	4-1.下を押したまま、Aを押して離す
		-> e
	4-2.左を押したまま、Aを押して離す
		-> h
		.. 最初に押す方向で、文字が変わる
		
	5.下を押したまま、下左にする、次にAを押して離す
		-> e
		.. さっきと同じように、斜めにすると次の文字へ
		
	6.上を押したまま、Bを押して離す
		-> o
		.. A か B かでも 文字が変わる
		
	7.上を押したまま、Aを押しつつ、Bを押す(クロス打ち)
		-> B
		.. クロス打ちで、大文字になる
	
	8.上を押したまま、なめらかに左にする、次にAを押して離す
		-> !
		.. 90度曲げると、記号が打てる
	
	(注) m, q のみ、少しわかりにくくなっており、
	記号と同じように
	< 上を押したまま、斜めでなく「右(左)」にする >
	必要があります。
