1.文字の打ち方
	十字キーの操作と、ABを組み合わせて文字を打ちます。
	(例)
	1.上を押したまま、Aを押して離す
		-> a
	2.上を押してから、上右にする、次にAを押して離す
		-> b
	3.上を押してから、上左にする、次にAを押して離す
		-> c
	4.右を押したまま、Aを押して離す
		-> d
	5.右を押したまま、右下にする、次にAを押して離す
		-> e
	...
	6.上を押したまま、Bを押して離す
		-> m
	7.上を押したまま、Aを押しつつ、Bを押す(クロス打ち)
		-> A (大文字)
	
	(注) p, q のみ、少しわかりにくくなっており、
		< 上を押したまま、斜めでなく「右(左)」にする >
		必要があります。
		
		アルファベットが26文字で、
		3方向 * 8方向 より 2つ はみ出てしまうので
		ここで折り合いをつけました。
